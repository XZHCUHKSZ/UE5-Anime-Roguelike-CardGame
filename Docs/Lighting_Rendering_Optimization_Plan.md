# 光照与渲染优化方案（UE5）

## 1. 目标性能档位
- 高配档（开发主机）：1080p / 60 FPS 稳定
- 中配档（目标主流）：1080p / 45-60 FPS
- 低配档（兜底）：900p / 30+ FPS

## 2. 光照策略
- 早期（功能开发期）：
  - 使用简单主方向光 + 天空光
  - 减少动态阴影灯数量
  - 临时关闭高开销后处理
- 中期（视觉打磨期）：
  - 关键场景启用更高质量阴影
  - 调整 Lumen 质量档，不盲目拉满
  - 对角色与敌人做重点补光
- 后期（发版期）：
  - 固定不同质量档（Low/Medium/High）
  - 验证每档实际帧率和视觉可接受性

## 3. 关键预算建议
- 同屏动态光源：尽量 <= 3
- 高成本粒子：战斗场景同时 <= 5 组
- 后处理：避免多层叠加高开销效果
- 半透明材质：严格控制面积和数量

## 4. 常用优化抓手
- 减少阴影级联距离
- 降低 Lumen 质量或反射质量档位
- 控制 Nanite / 非Nanite资产混用策略
- UI层减少高频全量刷新
- 降低过重的屏幕后处理链

## 5. 分阶段检查点
- 每周固定一次性能采样（战斗场景 + 地图场景）
- 每次新增大特效必须复测帧率
- 版本冻结前做全流程性能回归

## 6. 工具与指标
- `stat unit`, `stat gpu`, `stat fps`
- Unreal Insights（关键卡顿分析）
- 日志记录三类指标：
  - 平均FPS
  - 1% low FPS
  - 峰值帧耗时
