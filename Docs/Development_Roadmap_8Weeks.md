# 8周开发排期（纯单机）

## 第1-2周：战斗最小闭环
- 完成基础回合状态机：玩家回合 -> 敌方回合 -> 回合结算
- 实装卡牌基础类型：攻击、防御、增益
- 完成敌方意图显示（攻击/防御/上Buff）
- 可完成一场1v1战斗并结算胜负

## 第3-4周：内容填充1（可玩）
- 卡牌扩充到30-40张
- 敌人扩充到8-10个（含1精英）
- 完成基础地图节点（战斗、事件、商店、休息）
- 完成遗物系统V1（拾取 + 被动生效）

## 第5-6周：内容填充2（完整跑图）
- 卡牌扩充到60张
- 敌人12个 + Boss 2个
- 事件扩充到25个（分稀有度）
- 完成局内奖励选择、删卡、升级
- 可完整跑完一局（开局->Boss）

## 第7周：局外成长与存档
- 完成局外解锁：角色、卡牌、遗物条目
- 完成本地存档（局内/局外分槽）
- 失败与胜利结算接入Meta进度
- 回放必要信息记录（种子、关键战斗日志）

## 第8周：打磨与提测
- 数值首轮平衡（按目标回合数调参）
- 基础新手引导与UI可读性优化
- 性能与稳定性检查（崩溃、卡死、存档异常）
- Demo打包与自测清单闭环

## 每周固定里程碑输出
- 周中可玩构建（至少1个主流程）
- 周末测试报告（Bug、阻塞项、下周风险）
