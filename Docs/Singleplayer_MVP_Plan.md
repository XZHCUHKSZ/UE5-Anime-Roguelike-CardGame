# 单机二次元卡牌肉鸽（UE5）MVP方案

## 1. 项目目标（MVP）
- 类型：纯单机、回合制卡牌肉鸽、类杀戮尖塔
- 时间：8周可玩 Demo（可打完 1 条完整爬塔）
- 团队：1-3人
- 平台：PC（Windows）优先

## 2. 单机约束（强制）
- 全流程离线可玩，不依赖任何联网服务
- 不做多人、不做PVP、不做联机同步
- 账号系统缺省为无（可后续接入Steam云存档，但不是MVP阻塞项）

## 3. 首发内容量（MVP封顶）
- 可玩角色：2名
- 卡牌：60张（每角色20张专属 + 20张中立）
- 敌人：12个普通/精英 + 2个Boss
- 遗物：40个
- 事件：25个
- 地图层级：3大层 + 最终Boss层（总计约12-15战斗节点）

## 4. 玩法闭环（单局）
1. 角色选择（初始卡组 + 起始遗物）
2. 地图爬塔（战斗/事件/商店/休息）
3. 回合战斗（抽牌、打牌、能量管理、敌方意图）
4. 局内成长（拿卡、删卡、升级、遗物）
5. Boss结算（胜利解锁局外内容）

## 5. 系统拆分（UE5）
- `BP_GameInstance_Rogue`：全局进度加载、存档切换
- `BP_RunManager`：单局状态、随机种子、地图推进
- `BP_BattleManager`：回合状态机与结算流程
- `BP_CardSystemComponent`：抽牌堆/弃牌堆/手牌/耗能
- `BP_StatusComponent`：Buff/Debuff叠层与衰减
- `BP_EnemyIntentComponent`：敌方行为脚本与意图展示
- `BP_MetaProgression`：局外解锁（角色、卡、遗物）

## 6. MVP验收标准（DoD）
- 能完整开局到通关（或失败结算）且无阻塞Bug
- 本地存档可保存并读取“局内进度+局外解锁”
- 至少2个角色都能通关1次（平衡可粗糙）
- 60FPS目标下战斗UI可稳定操作（1080p）

## 7. 明确不做（避免超范围）
- 多人模式、实时联机、排行榜
- 复杂卡面动画工厂（先用简化演出）
- 语音全量实装（先做占位资源）
- 超大剧情线（先事件文本驱动）
